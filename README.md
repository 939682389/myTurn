# Gathering
一款线上活动组团微信小程序

## 接口文档
- http://gathering.cooltian.cn/v1/

## 运行环境
- uWSGI、Nginx
- MySQL >=5.1
- python >=3.5

## 功能
- 小组创建
- 活动创建
- 用户聊天
- 消息通知
